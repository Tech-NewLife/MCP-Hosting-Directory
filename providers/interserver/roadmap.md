# Roadmap

This roadmap outlines the vision and future direction for the InterServer MCP integration. It provides transparency into planned improvements, priorities, and long-term goals for developers building AI-powered hosting automation.

> **Note**
>
> This roadmap is for documentation purposes and reflects the goals of this repository. It should not be interpreted as the official product roadmap of InterServer. Features and priorities may evolve as the InterServer API and Model Context Protocol (MCP) ecosystem continue to grow.

---

# Vision

Build a secure, reliable, and developer-friendly MCP server that enables AI assistants and automation platforms to interact seamlessly with InterServer services through standardized MCP tools and natural language.

---

# Guiding Principles

- Security first
- Standards-compliant MCP implementation
- Stable APIs
- Developer-friendly documentation
- Automation-focused workflows
- High reliability
- Continuous improvement

---

# Current Capabilities

The InterServer MCP server currently supports:

- Account information
- Public MCP endpoints
- Private authenticated MCP endpoints
- Hosting account management
- VPS management
- Dedicated server management
- DNS operations
- Billing information
- Support ticket management
- SSL-related services
- Hosting automation through MCP-compatible clients
